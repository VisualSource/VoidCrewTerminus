Chat Commands & Item Spawning

VoidManager 1.2.10  ·  VoidCrew.GameLibs 1.3.0  ·  Assembly-CSharp (IL2CPP stubs)

Overview

VoidManager auto-discovers any class that inherits from ChatCommand or PublicCommand via reflection — no manual registration is needed. Item spawning is handled by SpawnUtils.SpawnCarryable, which calls into Photon to synchronise the spawned object across all connected clients.

ChatCommand vs PublicCommand

ChatCommand VoidManager.Chat.Router

Invoked with a leading /. Runs only on the local client.

MemberDescription
string[] CommandAliases()Trigger words. First entry is the primary name.
string Description()Shown in the in-game help list.
List<Argument> Arguments()Argument tree used for tab-completion.
string[] UsageExamples()Shown in detailed help. Default: /alias.
void Execute(string arguments)Called with everything typed after the alias, trimmed.
PublicCommand VoidManager.Chat.Router

Invoked with a leading !. Runs on all clients in the session.

MemberDescription
void Execute(string arguments, int sender)sender is the Photon actor number of the player who typed the command.
All other members are identical to ChatCommand.
Heads-up: Because Execute runs on every client, actions that mutate game state (like spawning items) should be gated to one client to avoid duplicates. Use PhotonNetwork.IsMasterClient or PhotonNetwork.LocalPlayer.ActorNumber == sender.

Argument Trees

Argument builds a tree that drives tab-completion. Each node holds one or more option names and optional child nodes.

Argument VoidManager.Chat.Router
ConstructorUse
new Argument(params string[] names)Leaf node — one or more completable options at this level.
new Argument(string name, Argument child)Node that leads to a single required sub-argument.
new Argument(string[] names, List<Argument> children)Node with multiple sub-argument branches.

Special placeholder names:

PlaceholderMeaning

%integer / %numberSignals a numeric input is expected.

Enumerating Spawnable Items

The game maintains a pre-built list of all spawnable assets in DebugSpawnObjects.SpawnablesList (CG namespace). Each entry exposes:

  • name — display name of the item
  • _guidUnion — the GUIDUnion to pass to SpawnCarryable
  • _objectType — System.Type for filtering (e.g. typeof(CarryableObject))

Call DebugSpawnObjects.PopulateList() lazily before first use. Filter to carryables with entry._objectType == typeof(CarryableObject) (CG.Objects namespace). 

using CG;
using CG.Objects;
using CG.Game.Scenarios;

if (DebugSpawnObjects.SpawnablesList.Count == 0)
    DebugSpawnObjects.PopulateList();

var carryables = DebugSpawnObjects.SpawnablesList
    .Where(s => s._objectType == typeof(CarryableObject))
    .ToList();

var item = carryables
    .FirstOrDefault(s => s.name.Contains(search, StringComparison.OrdinalIgnoreCase));

if (item != null)
    SpawnUtils.SpawnCarryable(item._guidUnion, position, Quaternion.identity);
Multiplayer: SpawnCarryable calls into Photon internally. In a PublicCommand (which runs on all clients), gate the spawn call to one client — e.g. if (PhotonNetwork.IsMasterClient) — to avoid N duplicates.

Full Example — !spawn command

Full source: VoidCrewTerminus/Commands/SpawnItem.cs

using System;
using System.Collections.Generic;
using System.Linq;
using CG;
using CG.Game.Player;
using CG.Game.Scenarios;
using CG.Objects;
using UnityEngine;
using VoidManager.Chat.Router;
using VoidManager.Utilities;

internal class SpawnItemCommand : PublicCommand
{
    public override string[] CommandAliases() => ["spawn"];
    public override string Description() => "Spawn a carryable item at your position for testing";

    public override List<Argument> Arguments() =>
    [new("list"), new("%item_name")];

    public override void Execute(string arguments, int sender)
    {
        if (string.IsNullOrWhiteSpace(arguments))
        {
            Messaging.Notification("Usage: !spawn <name>  |  !spawn list [filter]");
            return;
        }

        if (DebugSpawnObjects.SpawnablesList.Count == 0)
            DebugSpawnObjects.PopulateList();

        var carryables = DebugSpawnObjects.SpawnablesList
            .Where(s => s._objectType == typeof(CarryableObject)).ToList();

        if (arguments.StartsWith("list", StringComparison.OrdinalIgnoreCase))
        {
            var filter = arguments.Length > 4 ? arguments[4..].Trim() : string.Empty;
            var hits = string.IsNullOrEmpty(filter)
                ? carryables
                : carryables.Where(s => s.name.Contains(filter, StringComparison.OrdinalIgnoreCase)).ToList();
            for (int i = 0; i < hits.Count; i += 10)
                Messaging.Notification(string.Join(", ", hits.Skip(i).Take(10).Select(s => s.name)));
            return;
        }

        var player = LocalPlayer.Instance;
        if (player == null) { Messaging.Notification("Not in an active session"); return; }

        var item =
            carryables.FirstOrDefault(s => s.name.Equals(arguments, StringComparison.OrdinalIgnoreCase))
            ?? carryables.FirstOrDefault(s => s.name.Contains(arguments, StringComparison.OrdinalIgnoreCase));

        if (item == null)
        {
            Messaging.Notification($"No carryable found matching '{arguments}'. Use !spawn list to browse.");
            return;
        }

        var pos = player.transform.position + player.transform.forward * 2f;
        var spawned = SpawnUtils.SpawnCarryable(item._guidUnion, pos, Quaternion.identity);
        Messaging.Notification(spawned != null ? $"Spawned: {item.name}" : $"Failed: {item.name}");
    }
}
